# sing-box 一键代理管理脚本

基于 [sing-box](https://github.com/SagerNet/sing-box) 的 SOCKS5 + HTTP 双协议代理一体化管理脚本，支持安装、升级、卸载、服务管理、连通性测试等全部操作，单文件无依赖。

---

## 功能概览

| 分类 | 功能 |
|------|------|
| 安装管理 | 交互式向导安装、升级到最新版、一键卸载 |
| 代理协议 | SOCKS5 + HTTP 双协议同时开启 |
| 认证支持 | SOCKS5 / HTTP 可独立配置用户名密码 |
| 服务控制 | 启动 / 停止 / 重启 / 查看状态 |
| 代理测试 | 连通性、延迟基准、认证验证、完整诊断 |
| 快捷命令 | SSH 直接输入 `x` 或 `X` 打开菜单 |
| 自动开机 | systemd 服务注册，崩溃自动重启 |
| 防火墙 | 自动识别 ufw / firewalld 并放行端口 |

---

## 快速开始

```bash
# 下载脚本
wget -O singbox.sh https://your-server/singbox.sh

# 运行（需要 root）
sudo bash singbox.sh
```

首次运行会自动弹出提示，询问是否设置 `x` / `X` 快捷命令。设置后 SSH 登录任意目录直接输入：

```bash
x    # 或大写
X
```

即可打开管理菜单。

## 安装向导说明

选择菜单 `1` 后进入交互式配置向导，根据提示填写：

```
SOCKS5 端口 [默认 1080]:
HTTP   端口 [默认 1081]:
SOCKS5 用户名 [留空=不启用认证]:
SOCKS5 密码:
HTTP   用户名 [留空=不启用认证]:
HTTP   密码:
允许局域网访问? [y/N]:
```

> 所有选项均有默认值，直接回车即可使用最简配置。

## 代理测试说明

### 快速连通性测试（选项 8）
检测 SOCKS5 和 HTTP 代理是否可用，并显示出口 IP 和延迟：

```
SOCKS5   127.0.0.1:1080   ✓ 连通  出口 IP: 1.2.3.4  延迟: 128ms
HTTP     127.0.0.1:1081   ✓ 连通  出口 IP: 1.2.3.4  延迟: 135ms
```

延迟颜色评级：**绿色** < 200ms　**黄色** < 600ms　**红色** ≥ 600ms

### 延迟基准测试（选项 9）
对多个目标各测 3 次，取平均值：

```
▶ SOCKS5 代理
  ifconfig.me                         142ms  (3/3 成功)
  www.cloudflare.com                  98ms   (3/3 成功)
  httpbin.org/ip                      201ms  (2/3 成功)
```

### 认证配置验证（选项 10）
验证正确凭据能通过、错误凭据被拒绝：

```
▶ SOCKS5
  正确凭据 → 应成功         ✓ 通过
  错误凭据 → 应拒绝         ✓ 已拒绝
```

### 完整诊断报告（选项 11）
一次性检查服务状态、端口监听、连通性、认证配置、配置文件路径。

## 快捷命令

首次运行会弹出设置提示：

```
╔══════════════════════════════════════════════════════╗
║           设置快捷命令                               ║
╠══════════════════════════════════════════════════════╣
║  设置后，SSH 登录任意目录直接输入（大小写均可）：    ║
║                                                      ║
║      x   或   X                                      ║
║                                                      ║
║  即可打开此管理菜单，无需记住脚本路径。              ║
╠══════════════════════════════════════════════════════╣
║  Y  立即设置  (推荐)                                 ║
║  N  本次跳过                                         ║
║  S  不再提示                                         ║
╚══════════════════════════════════════════════════════╝
```

选择 `Y` 后脚本会复制自身到：

```
/usr/local/bin/x    ← 小写
/usr/local/bin/X    ← 大写
```

后续也可通过菜单选项 `14` 管理（更新 / 移除）。

## 常用命令速查

```bash
# 打开管理菜单
x

# 查看服务状态
systemctl status sing-box

# 查看实时日志
journalctl -u sing-box -f

# 重启服务
systemctl restart sing-box

# 手动验证 SOCKS5
curl --socks5 127.0.0.1:1080 https://ifconfig.me

# 手动验证 HTTP
curl --proxy http://127.0.0.1:1081 https://ifconfig.me
```

## 升级说明

菜单选项 `2` 会自动：

1. 查询 GitHub Releases 最新版本
2. 与当前版本对比，相同则提示无需升级
3. 显示版本变化（如 `v1.8.0 → v1.9.2`）并二次确认
4. 下载新版本，停止服务，替换二进制，重启服务

全程不修改配置文件，升级前后代理设置保持不变。
